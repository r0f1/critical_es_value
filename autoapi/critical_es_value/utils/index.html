critical_es_value.utils.get_alpha(confidence: float, alternative: str) → float¶

Calculate the significance level (alpha) corresponding to a given confidence level.

\[\begin{split}\alpha = \begin{cases} 1 - \text{conf} & \text{one-sided} \\ \frac{1 - \text{conf}}{2} & \text{two-sided} \end{cases}\end{split}\]
Parameters:

  • confidence (float) – Confidence level between 0 and 1 (exclusive).

  • alternative (str) – The alternative hypothesis. Either “two-sided”, “greater”, or “less”.

Returns:

The significance level (alpha).

Return type:

float

Raises:

  • ValueError – If confidence is not in (0, 1)

  • ValueError – If alternative is not one of “two-sided”, “greater”, or “less”.

Examples

>>> get_alpha(0.95, "two-sided")
0.025
>>> get_alpha(0.95, "less")
0.05
critical_es_value.utils.get_bias_correction_factor_J(dof: int) → numpy.float64¶

Calculate the bias correction factor J for Hedges’ g.

\[J(x) = \frac{\Gamma\left(\frac{x}{2}\right)}{\sqrt{\frac{x}{2}}\Gamma\left(\frac{x-1}{2}\right)}\]
Parameters:

dof (int) – Degrees of freedom.

Returns:

The bias correction factor J.

Return type:

np.float64

Examples

>>> get_bias_correction_factor_J(10)
0.92274560805
>>> get_bias_correction_factor_J(20)
0.96194453374
